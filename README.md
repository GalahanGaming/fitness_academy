# Fitness Academy System

A web-based gym management system built using:

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User registration/login
- Membership management
- Payment tracking
- Subscription management

## Requirements
- XAMPP or WAMP
- PHP
- MySQL

## Setup
1. Import the SQL file from the database folder
2. Place project inside htdocs
3. Start Apache and MySQL
4. Open localhost/project-folder
